# Muad Store - E-Commerce Platform (Vanilla JS)

A modern, responsive e-commerce web application built entirely with **Vanilla JavaScript, HTML5, and CSS3**. This project demonstrates a full shopping experience without the need for complex frameworks.

![Project Preview](img/logo.png)

## 🚀 Live Demo
You can view the project in action here: [Live Link](https://ecommerce-react-js-mocha.vercel.app/) 

## ✨ Key Features

- ** Interactive Shopping Cart**: Add and remove items, adjust quantities, and see real-time subtotal updates.
- ** Favorites / Wishlist**: Save your favorite products for later viewing.
- ** Advanced Search**: Filter products by name and category across the entire store.
- ** Fully Responsive Design**: Optimized for mobile, tablet, and desktop screens.
- ** Category Browsing**: Navigate through specialized categories like Electronics, Mobiles, and Appliances.
- ** Product Details**: In-depth view for each product including images, specifications, and availability.
- ** Checkout System**: Complete order summary with shipping calculation and coupon code support.
- ** Dynamic Sliders**: Featured products and hot deals displayed using Swiper JS.
- ** Persistence**: Cart and Favorites data are saved locally using `localStorage`.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Custom styling with Flexbox, Grid, and smooth transitions.
- **JavaScript (ES6+)**: Core business logic, DOM manipulation, and data fetching.
- **Swiper JS**: For touch-friendly product sliders and carousels.
- **Font Awesome**: High-quality icons for a premium UI feel.
- **JSON API**: Mocked product data served via `products.json`.

## 📁 Project Structure


## 🛠️ Setup and Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Muad-Ahmed/ecommerce-vanilla-js.git
   ```

2. **Navigate to the project folder**:
   ```bash
   cd "My store"
   ```

3. **Run the project**:
   - Simply open `index.html` in your favorite browser.
   - Alternatively, use a local server like **Live Server** in VS Code to avoid any CORS issues with the JSON fetch.
